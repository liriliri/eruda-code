## Demo

Browse it on your phone: 
[https://eruda.liriliri.io/?plugin=code](https://eruda.liriliri.io/?plugin=code)

## Install

```bash
npm install eruda-code --save
```

```javascript
eruda.add(erudaCode);
```

Make sure Eruda is loaded before this plugin, otherwise won't work.
